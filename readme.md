# My Photographic Site

## Author
**Tabitha Mburu**

## Project Description
My Photographic Site is a personal portfolio website that showcases a collection of my favorite photography work. The site features a responsive gallery, a short biography section, and a clean visual layout designed to provide a professional and user-friendly experience.  

This project was created as part of my learning journey in web development, focusing on HTML structure, CSS styling, and basic JavaScript interaction. It also demonstrates version control using Git and hosting via GitHub Pages.

## Features
- Fully responsive photo gallery with hover effects and lightbox previews  
- Short biography section about the photographer  
- Clean, modern design using HTML and CSS  
- Organized folder structure for easy maintenance  
- Interactive elements styled with custom CSS
